蓝牙搜索

蓝牙的搜索有三种方式

  • 方法一：BluetoothAdapter.startDiscovery()是通用的扫描方法，扫描时间持续10秒后会自动停止；当扫描到蓝牙设备后，会发出广播通知； 蓝牙扫描广播接收器

//注册此广播，监听BluetoothDevice.ACTION_FOUND，以接收系统消息取得扫描结果
private class DeviceReceiver extends BroadcastReceiver {
    @Override
    public void onReceive(Context context, Intent intent) {
        String action = intent.getAction();
        if(BluetoothDevice.ACTION_FOUND.equals(action)){
            BluetoothDevice device = intent.getParcelableExtra(BluetoothDevice.EXTRA_DEVICE);  
        }
    }
}
  • 方法二：BluetoothAdapter.startScan(ScanCallback callback)在ScanCallback的onScanResult(int callbackType, ScanResult result)方法中，通过result.getDevice()可获取扫描到的Ble设备
  • 方法三：BluetoothAdapter.startLeScan(BluetoothAdapter.LeScanCallback callback) 该方法已经被舍弃
