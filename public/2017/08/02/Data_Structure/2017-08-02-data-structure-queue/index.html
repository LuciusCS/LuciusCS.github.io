队列的表示与实现

modified: 2017-08-02
队列是先进先出的（FIFO）的线性表，在具体应用中通常使用链表或者数组实现；只允许在一端插入，在另一端删除；在队列中允许插入的一端叫队尾，允许删除的一端叫队头；

链队列——队列的链式表示和实现

链队列的本质是一个单向链表，头结点front相当于head,再增加一个尾结点。
