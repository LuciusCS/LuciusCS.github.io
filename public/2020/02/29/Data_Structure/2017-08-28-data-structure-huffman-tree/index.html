Huffman树的算法实现

modified 2017-08-27

Huffman树

Huffman树，又称最优二叉树或哈夫曼树，是一类加权路径长度最短的二叉树；

Huffman算法

设给定的权重集合为{7，5，2，4，6}，构造Huffman树的过程如下图所示。首先构造每棵树只有一个结点的森林，然后每次选择两个根结点权重最小的二叉树，以它们为左、右子树构造新的二叉树，最后得到一棵二叉树
